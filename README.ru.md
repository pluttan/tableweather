# TableWeather

**Настольный куб погоды — живые погодные сцены в прозрачном корпусе**

</div>

Настольное устройство, воссоздающее живые погодные сцены внутри прозрачного куба. Светодиоды, водяная помпа и парогенератор создают интерактивные световые и атмосферные эффекты, отражающие текущую погоду на улице.

## ■ Возможности

- ❖ **Прозрачный корпус-куб** — герметичный «аквариум», 3D-печатные стенки и отдельная подставка
- ❖ **Светодиодная подсветка** — LED-модули имитируют цвет и интенсивность неба при разных погодных состояниях
- ❖ **Эффекты воды и тумана** — помпа и парогенератор воспроизводят дождь и туман внутри куба
- ❖ **Кастомные 3D-модели** — стенки корпуса (TW.E.1.1) и подставка (TW.E.1.2) спроектированы в 3D-САПР

## ■ Как это работает

```
1. Герметичный прозрачный куб (3D-печатные стенки TW.E.1.1 и подставка TW.E.1.2) образует физический корпус.
2. LED-модули внутри куба имитируют цвет и интенсивность неба в соответствии с текущим погодным состоянием.
3. Водяная помпа и парогенератор активируются для воспроизведения эффектов дождя и тумана внутри корпуса.
```

## ■ Использование

```bash
# Открыть 3D-модели в совместимом САПР-редакторе
# аквариум.a3d.bak                        — полная модель корпуса аквариума
# Стенки аквариума _ TW.E.1.1.m3d.bak    — модель стенок корпуса
# Подставка _ TW.E.1.2.m3d.bak           — модель подставки

# Ознакомиться с документацией проекта
# ПОГОДА 2023.docx  — техническое задание и план работ
# ПОГОДА 2023.xlsx  — таблица данных проекта
```
