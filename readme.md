# TableWeather

**Desktop weather cube — real weather scenes in a transparent enclosure**

</div>

A desktop device that recreates live weather scenes inside a transparent cube. LEDs, a water pump, and a steam generator produce interactive lighting and atmospheric effects that reflect current outdoor conditions.

## ■ Features

- ❖ **Transparent cube enclosure** — sealed "aquarium" body, 3D-printed walls and a dedicated stand
- ❖ **LED lighting** — LED modules simulate sky color and intensity for different weather states
- ❖ **Water & mist effects** — pump and steam generator reproduce rain and fog atmosphere inside the cube
- ❖ **Custom 3D models** — enclosure walls (TW.E.1.1) and stand (TW.E.1.2) designed in 3D CAD

## ■ How It Works

```
1. A sealed transparent cube (3D-printed walls TW.E.1.1 and stand TW.E.1.2) forms the physical enclosure.
2. LED modules inside the cube simulate sky color and intensity to match the current weather state.
3. A water pump and steam generator activate to reproduce rain and fog effects within the enclosure.
```
